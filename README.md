# Deep-JS-Foundations

These excercise files are taken and completed from the course "Deep JS Foundation" by Kyle Simpson.
Workshop held by frontEndMasters. https://frontendmasters.com/courses/deep-javascript-v3

These course covers: 

- Types & Coercion
    - Primitive Types.
    - Natives.
    - Coercion. 
    - Equality
    
- Scope, Closures
    - Nested Scope
    - Hoisting
    - Closure
    - Modules

- Objects
  - The this keyword
  - The new keyword
  - ES6 class keyword
  
- Prototype
  - The prototype chain
